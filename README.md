# Waveback RN

This repository contains one app: [Waveback RN](./waveback-rn), an Expo + React Native music-era player interface styled with Tailwind through NativeWind.

## Run on macOS

Open Terminal in the repository folder and run:

```sh
cd waveback-rn
npm install
npm start
```

Open the URL shown in the terminal, usually `http://localhost:8081`.

## Run on Windows

Open PowerShell in the repository folder and run:

```powershell
cd waveback-rn
npm.cmd install
npm.cmd start
```

Open the URL shown in the terminal, usually `http://localhost:8081`.
